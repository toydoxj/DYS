# DYS - 모듈화된 구조 설계 애플리케이션

## 새로운 모듈화 구조

이 디렉토리는 기존 `UI/DYL.py`의 거대한 단일 파일을 모듈화하여 재구성한 버전입니다.

## 구조

```
dys/
├── main.py                    # 애플리케이션 진입점
├── config/                    # 설정 관리
│   ├── constants.py          # 상수 정의
│   └── settings.py           # 애플리케이션 설정
├── core/                      # 핵심 비즈니스 로직
│   ├── models/               # 데이터 모델
│   │   ├── base.py           # 기본 모델 클래스
│   │   ├── project.py       # 프로젝트 정보 모델
│   │   └── load.py           # 하중 데이터 모델
│   └── services/            # 비즈니스 로직 서비스
│       └── file_service.py   # 파일 관리 서비스
├── ui/                        # UI 관련
│   ├── main_window.py        # 메인 윈도우
│   ├── widgets/              # 재사용 가능한 위젯
│   │   ├── left_menu.py     # 좌측 메뉴
│   │   └── header.py        # 헤더
│   └── pages/                # 각 페이지 (추가 예정)
└── utils/                     # 유틸리티
    ├── file_io.py            # 파일 I/O
    ├── csv_loader.py         # CSV 로더
    └── validators.py        # 입력 검증
```

## 실행 방법

```bash
python dys/main.py
```

## 주요 개선사항

1. **모듈화**: 거대한 단일 파일을 기능별로 분리
2. **MVC 패턴**: 모델, 뷰, 컨트롤러 분리
3. **서비스 레이어**: 비즈니스 로직을 서비스로 분리
4. **재사용성**: 위젯과 유틸리티 모듈화로 재사용 가능
5. **테스트 가능**: 각 모듈을 독립적으로 테스트 가능
6. **유지보수성**: 코드 구조가 명확하여 유지보수 용이
